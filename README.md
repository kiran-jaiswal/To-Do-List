# 📝 To-Do-List

A simple and clean **To-Do List** web application built with **Spring Boot** and **Java**.

---

## 🚀 Tech Stack

- **Java** – Core programming language
- **Spring Boot** – Backend framework
- **Spring MVC** – Web layer (REST / MVC pattern)
- **Maven** – Dependency management and build tool
- **Thymeleaf / REST API** – Frontend rendering or API endpoints

---

## ✨ Features

- ✅ Add new tasks
- 📋 View all tasks
- ✏️ Update existing tasks
- 🗑️ Delete tasks
- 🔄 Mark tasks as complete / incomplete

---

## ⚙️ Prerequisites

Make sure you have the following installed:

- **Java 17+** (or whichever version you used)
- **Maven 3.6+** *(or use the included Maven wrapper)*

---

## 🛠️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/kiran-jaiswal/To-Do-List.git
cd To-Do-List
```

### 2. Run with Maven Wrapper

**Linux / Mac:**
```bash
./mvnw spring-boot:run
```

**Windows:**
```cmd
mvnw.cmd spring-boot:run
```

### 3. Open in browser

```
http://localhost:8080
```

---

## 🔧 Build the Project

```bash
./mvnw clean install
```
